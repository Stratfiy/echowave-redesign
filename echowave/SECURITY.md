# Security Policy

At EchoWave we look forward to working with security researchers across the world to keep EchoWave and our users safe. If you have found an issue in any of our services, please reach out to us.

## Supported Versions
We always recommend using the latest versions of services to ensure you get all security updates

## Reporting a Vulnerability

If you believe you have found a security vulnerability within EchoWave, please let us know right away. We'll try and fix the problem as soon as possible.

**Do not report vulnerabilities using public GitHub issues**. Instead, report the issue [here](https://github.com/echowave-hq/echowave/security/advisories/new)

Once we've received your email we'll keep you updated as we fix the vulnerability.

## Thanks

Thank you for keeping EchoWave and our users safe. 🙇
